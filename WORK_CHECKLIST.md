# AI Game Blind Arena Unified Checklist

이 문서는 현재 코드 기준의 단일 체크리스트입니다.
기존 `WORK_CHECKLIST.md`와 `DEPLOY_NOTES.md`를 통합했고, 구현과 맞지 않던 오래된 항목은 제거하거나 현행화했습니다.

표기 규칙:
- `[x]` 현재 코드 기준으로 구현 완료 또는 정리 완료
- `[ ]` 배포/운영 전 사람이 직접 결정하거나 확인해야 하는 항목

기준 시점:
- 프런트: `backend/static`
- 백엔드: `backend/main.py`, `backend/utils.py`
- DB 스키마: `backend/schema.sql`
